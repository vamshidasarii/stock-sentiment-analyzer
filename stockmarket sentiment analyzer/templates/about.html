About This Project

The Stock Market Sentiment Analyzer is a real-time web application built to help users understand the market sentiment surrounding various companies. Using advanced Natural Language Processing (NLP) tools, this platform analyzes live news headlines and classifies them into positive, neutral, or negative sentiments. This enables traders, investors, and market enthusiasts to make better decisions based on current media narratives.

The heart of this system lies in its use of the VADER Sentiment Analysis model from the NLTK library. News is fetched using the NewsAPI service, processed in the backend with Python and Flask, and displayed to the user through a clean and user-friendly interface using HTML, CSS, and JavaScript.

Whether you're a beginner or an expert in stock trading, this tool aims to simplify market insights and make sentiment data accessible to everyone. Built with the goal of demonstrating the power of real-time data processing and sentiment tracking, this project is a perfect example of how technology meets finance.
