Stock Market Sentiment Analyzer

This project is a real-time web-based application designed to analyze and visualize the sentiment of recent news headlines related to a company or stock. It uses natural language processing (NLP) to determine whether news articles are expressing positive, negative, or neutral sentiment. The goal is to help users — especially traders, investors, and analysts — make smarter decisions based on media trends and market emotion.

⚙️ Technologies Used:

Flask • Python • NLTK VADER • NewsAPI • HTML • CSS • JavaScript

🔗 Data Source: NewsAPI.org | Sentiment Analysis: NLTK VADER
